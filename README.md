# SoulSpace 💚

A free, anonymous peer support platform for random 1-on-1 chats.

**Key features:**
- Completely anonymous
- Random matching
- Ephemeral chats (messages delete when session ends)
- Calm, safe design
- Built for free with Turso + Next.js

## Tech Stack
- Next.js 16 + TypeScript + Tailwind
- Turso (LibSQL) + Drizzle ORM
- Ephemeral chat (no permanent message storage)

## Getting Started

1. Clone the repo
2. Create a Turso database (free): https://turso.tech
3. Add your Turso credentials to `.env.local`
4. Run `npm run db:push`
5. `npm run dev`

Made with ❤️ for mental support.
